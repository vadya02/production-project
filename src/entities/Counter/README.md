## Сущность счетчика

Описание:
сущность для работы с простым счетчиком.

#### Public api

- Components

`Counter` - компонент счетчика с кнопками увеличения и уменьшения

- types

`CounterSchema` - схема состояния счетчика

- reducers

`counterReducer` - редьюсер для управления состоянием счетчика
